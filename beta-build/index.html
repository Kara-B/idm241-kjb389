Trigger

  • Cursor hovers over recipe card
  • Cursor hovers over save recipe icon
  • User clicks save recipe icon
  • Cursor hovers off of save recipe icon
  • Cursor hovers off of recipe card

Rules

On Hover Card

  • Cursor turns to pointer
  • Save recipe icon fades in for .5s

On Hover over Save Recipe Icon

  • The text prompt “save” fades in for .5s below the icon
  • A small red box shadow fades in behind the icon for .5 seconds

Click Save Recipe Icon

  • Bookmark stretches down from the top of the card for 1s to cover the save icon
  • Once covered, icon background color immediately changes to red. At the same time, Icon changes to white
  • Once covered, text prompt changes from save, to saved
  • Bookmark snaps back to the top to the top of the card for .5s
  • The recipe is added to the users list of saved recipes
  • Clicking the icon a Second time will result in the inverse of the above steps

On Hover Off Save Recipe Icon

  • The text prompt “save” fades out for .5s below the icon
  • The red box shadow on the icon fades away for .5 seconds

On Hover Off Recipe Card

  • Pointer turns to cursor
  • Save recipe icon fades out for .5s
